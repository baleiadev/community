# ⚛️ Laboratórios Interativos de Física e Óptica

Este diretório contém simuladores interativos desenvolvidos com HTML, CSS e JavaScript para explorar conceitos fundamentais de óptica física, propagação de ondas e comunicações ópticas.

## 🔬 Simuladores Disponíveis

* **Interferência e Ondas:** Anéis de Newton, Interferômetro de Michelson, Fabry-Pérot.
* **Polarização:** Lei de Malus, Fotoelasticidade, Placas de Meia-Onda.
* **Refração e Dispersão:** Dispersão, Películas Antirreflexo, Decomposição do Espectro.

## 🚀 Como utilizar

Basta clonar o repositório e abrir o arquivo `labs-fisica.html` no seu navegador para acessar o menu principal com todas as simulações, não é necessário rodar nenhum servidor local.
